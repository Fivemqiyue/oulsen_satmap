# oulsen_satmap
# 高清的邮政地图-Fivem插件

将 oulsen_satmap 文件夹上传到服务器上的资源文件夹

将 oulsen_satmap 添加到 server.cfg 文件（启动或确保）。

启动资源或重新启动服务器!
